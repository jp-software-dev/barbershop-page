# Barbershop Web Platform

Professional full-stack platform for a barbershop, featuring a complete landing page, an integrated online store, and a dedicated Admin Dashboard.

## Features

* **Landing Page:** Complete public-facing site including Home, Services, Location, and Contact sections.
* **E-commerce Store:** Integrated online shop for grooming products.
* **Admin Dashboard:** Secure administrative module for business management.
* **Inventory Control (CRUD):** Full Create, Read, Update, and Delete capabilities for real-time product management in the store.

## Technologies Used

* **Frontend:** HTML5, CSS3, JavaScript
* **Architecture:** Full-stack integration with CRUD operations
