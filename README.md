Professional web platform designed for a barbershop. This project features a complete public-facing landing page, an integrated e-commerce store for grooming products, and a dedicated, secure Admin Dashboard for business management.

## Features

* **Public Landing Page:** Engaging interface including Home, Services, Location, and Contact sections to attract and inform clients.
* **E-commerce Store:** A fully integrated online shop allowing users to browse grooming products.
* **Admin Dashboard:** A secure, restricted-access administrative module.
* **Inventory Control (CRUD):** Real-time product management capabilities allowing administrators to Create, Read, Update, and Delete store items seamlessly.

## Technologies Used

* **Frontend:** HTML5, CSS3, Vanilla JavaScript.
* **Data Management:** Client-side storage (LocalStorage/SessionStorage) for handling authentication and CRUD operations in the admin panel without a backend.
